🚀 Kişisel Portföy Web Sitesi
Bu proje, benim bilgisayar mühendisliği öğrencisi olarak kendimi, yeteneklerimi ve projelerimi tanıtmak amacıyla geliştirdiğim ilk portföy sitemdir. Projeyi yaklaşık 2.5 günde hazırladım. Modern web tasarım trendlerinden olan Glassmorphism estetiğiyle geliştirilmiştir.

✨ Özellikler
Modern Tasarım: CSS Glassmorphism ve Flexbox kullanılarak estetik bir görünüm sağlandı.

Responsive Tasarım: Media Queries kullanılarak mobil, tablet ve masaüstü cihazlar için tam uyumlu hale getirildi.

Hızlı ve Optimize: Saf HTML5 ve CSS3 (Vanilla) kullanılarak, kütüphane bağımlılığı olmadan geliştirildi.

Etkileşimli Arayüz: CSS Hover efektleri ve yumuşak geçişler (transitions) ile kullanıcı deneyimi artırıldı.

🛠 Kullanılan Teknolojiler
HTML5: Semantik yapı ve içerik yönetimi.

CSS3: Flexbox, Media Queries, Pseudo-elements (::before), Backdrop-filter.
